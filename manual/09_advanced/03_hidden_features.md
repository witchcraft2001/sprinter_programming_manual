# 9.3 Скрытые возможности и специальные регистры

> **Навигация:** [← 9.2 Turbo](02_turbo.md) | [Оглавление](../README.md) | [9.4 ZX совместимость →](04_zx_compat.md)

---

## Обзор

В Sprinter есть несколько "экспертных" функций, которые не покрываются стандартными учебниками, но встречаются в продвинутом коде (SDK, демо, системные утилиты). Эти функции появляются в исходниках FPGA (`SP2_ACEX.TDF`, `DCP.TDF`) и иногда в BIOS.

---

## SC — System Configuration register

Регистр `SC[7..0]` управляется записью в порты `#C0`/`#C8` (DCP паттерн `B"1100X000"`). Биты:

| Бит | Имя | Назначение |
|-----|-----|-----------|
| 0 | SC0 | ZX режим: 1 = ZX Spectrum совместимость |
| 1 | SC1 | ROM16 режим |
| 2 | SC2 | NMI разрешение |
| 3 | SC3 | Дополнительный режим |
| 4 | SC4 | BLK_R — доступ к расширенной памяти |

### Включение/выключение ZX режима

```asm
; Включить ZX-совместимость
        ld      a, #01              ; SC0 = 1
        out     (#C0), a

; Выключить (нативный Sprinter)
        xor     a
        out     (#C0), a
```

---

## Конфигурационный порт #7F / #3F

Порт `#7F` (запись через `SYS_PORT_ON = #7C`) управляет глобальной конфигурацией машины. Побитовая раскладка (из `X_PLAN.TXT`):

| Бит | Имя | Назначение |
|-----|-----|-----------|
| 0 | TURBO on/off | Включение/выключение турбо-режима |
| 1 | TURBO ena | Разрешение переключения турбо |
| 2 | CNF ena | Разрешение записи конфигурации |
| 3 | CNF0 | Бит конфигурации 0 |
| 4 | CNF1 | Бит конфигурации 1 |
| 5 | CNF_PN_RESET | Сброс Pentagon-портов (по 1) |
| 6 | CNF_SC_RESET | Сброс Scorpion-портов (по 1) |
| 7 | CNF_PN512_RESET | Сброс Pentagon-512 (по 0) |

```asm
SYS_PORT_ON     EQU #7C
SYS_PORT_OFF    EQU #3C

; Включить системный режим
        ld      a, 0
        out     (SYS_PORT_ON), a

; Выключить
        ld      a, 0
        out     (SYS_PORT_OFF), a
```

Порты `#7C`/`#3C` также управляют доступом к системным страницам (`#FE`, `#FC`).

---

## Сдвиг экрана

Регистр сдвига позволяет программно смещать изображение на экране:

| Биты | Ось | Диапазон |
|------|-----|---------|
| [3:0] | Горизонталь | 0 = лево, 7 = центр, 14 = право, 15 = нет H-sync |
| [7:4] | Вертикаль | 0 = верх, 7 = центр, 14 = низ, 15 = нет V-sync |

> **Внимание:** значение 15 (по обеим осям) **отключает синхроимпульс** — на экране будет «каша». Используйте только значения 0–14.

---

## CNF — Configuration register

Управляется через `SYS_ENA` в DCP.TDF. Биты:

| Бит | Имя | Назначение |
|-----|-----|-----------|
| 3 | CNF3 | Scorpion режим |
| 4 | CNF4 | ROM16 on/off |
| 5 | CNF5 | Сброс PN5 |
| 6 | CNF6 | Scorpion-OFF |
| 7 | CNF7 | Pentagon-512 |

Запись в CNF через DCP системный механизм (обычно только BIOS).

---

## PN — Page Number register

Управляет дополнительными битами номера страницы для совместимости с Pentagon 512 и Scorpion:

| Бит | Назначение |
|-----|-----------|
| 0–3 | Дополнительные биты номера страницы (Scorpion) |
| 5 | PN5 — Pentagon-512 |
| 6 | PN6 — дополнительный |
| 7 | PN7 — ROM-RAM переключение |

---

## RGMOD — видеорегистр

Управляется через `OUT (#C9), A` или `OUT (#CD), A` (DCP паттерн `B"1100X101"`). Из `SP2_ACEX.TDF`:

```
SVIDEO.DIR_PORT[3] = RGMOD0;            -- бит 0 регистра RGMOD → выбор экрана
SVIDEO.DIR_PORT[4] = RGMOD1;            -- бит 1 регистра RGMOD → screen off/on
```

| Бит | Назначение |
|-----|-----------|
| 0 | Double buffer flip — выбор активной видеостраницы (A / B) |
| 1 | Screen off: 1 = экран выключен, 0 = включён |
| [7:2] | Не используются |

> Имена `RGMOD0`/`RGMOD1` в исходниках FPGA — это буквально «бит 0 / бит 1 регистра RGMOD», а не отдельные контакты. Обратите внимание, что в `SP2_ACEX.TDF` они попадают в разряды 3 и 4 сигнала `DIR_PORT[]` внутри видеоконтроллера — это **внутренние линии FPGA, а не биты 3/4 порта `#C9`**.

---

## HOLD — регистр управления акселератором

Управляется через `OUT (#CB), A` (DCP `B"11001011"`):

```asm
; Запись HOLD обычно выполняется SDK или компилятором для синхронизации
HOLD    EQU #CB

        ld      a, hold_value
        out     (HOLD), a
```

Используется для точного контроля за состоянием акселератора при критичных операциях.

---

## ALL_MODE — режим графики

Управляется через `OUT (#C3), A`:

| Значение | Режим |
|---------|-------|
| `#00` | ZX (256×192) |
| `#01` | 320×256, 8bpp |
| `#02` | 640×256, 4bpp |
| `#80` | Pentagon 128 compat |

```asm
        ld      a, #01
        out     (#C3), a            ; 320×256
```

---

## FN_ACC — функция акселератора (XMD)

Специальные режимы для блочных операций с маскированием:

| FN_ACC | Операция | Опкод-триггер |
|--------|---------|---------------|
| 0 | `XMD = MD` (копирование) | `LD D,(HL)` = `#56` |
| 1 | `XMD = MD OR RAM` | `#66` |
| 2 | `XMD = MD XOR RAM` | `#46` |
| 3 | `XMD = MD AND RAM` | `#6E` |

Активация через специальные опкоды `10XXXXXX`. Используется для прозрачных спрайтов и эффектов наложения.

---

## ALT_ACC — альтернативный режим акселератора

Переключает акселератор в альтернативный режим работы с памятью:

```asm
; Запись в порт #C7 (DCP `B"1100X111"`) активирует ALT_ACC
        ld      a, value
        out     (#C7), a
```

Детали использования выходят за рамки стандартной документации.

---

## VGA_SWITCH

Программное переключение между SCART и VGA выходами (если поддерживается):

```asm
; Записать в специальный порт MAX7000 для выбора видеовыхода
; (конкретный порт зависит от прошивки)
```

На большинстве плат Sprinter это физический переключатель, а не программный.

---

## DOS флаг

Внутренний сигнал `DOS` в FPGA определяет, находимся ли мы в "DSS окружении". Это влияет на поведение нескольких DCP-декодов. Программно переключается через определённые значения порта `#1F` (WR_PDOS).

---

## Soft Reset

Программный сброс системы:

```asm
; Warm reset через BIOS
        jp      #0000               ; перезапуск с начала ROM

; Cold reset (полный) — через Flash перепрошивку или аппаратный сброс
```

DCP адрес `20 : 7160` = Soft Reset через MAX7000.

---

## NMI

Non-Maskable Interrupt. В Sprinter NMI может быть активирован:
- Аппаратной кнопкой (Magic button)
- Сигналом `NMI_SW` через FPGA

NMI всегда переходит на `RST #66` (адрес `0x0066`).

```asm
; Обработчик NMI
        org     #0066
        push    af
        ; ... обработка NMI ...
        pop     af
        retn                        ; возврат из NMI
```

В нормальной работе NMI обычно отключён (`NMI_ENA = 0`).

---

## Специальные страницы памяти

Некоторые номера страниц зарезервированы для особых случаев:

| Страница | Назначение |
|---------|-----------|
| `#FE` | Системная DSS (резидент, переменные) |
| `#FD` | COVOX DMA буфер (при активном DMA) |
| `#FC` | Дополнительная системная |
| `#40`–`#4F` | Системные страницы DSS |
| `#50`–`#5F` | VRAM |
| `#80`–`#FF` | ROM / Flash |

---

## Tape port (#FE)

Стандартный ZX Spectrum tape input/output через порт `#FE`:

- **Бит 6** (чтение) — tape in
- **Бит 3** (запись) — tape out (MIC)
- **Бит 4** (запись) — EAR (speaker)

Эмулируется FPGA для совместимости с ZX программами, но в реальных платах Sprinter магнитофонный вход редко используется.

---

## Ключевые моменты

> - SC, CNF, PN — регистры управления режимами совместимости и банкинга.
> - `SYS_PORT_ON #7C` / `SYS_PORT_OFF #3C` — включение системного режима.
> - RGMOD, ALL_MODE — видеорегистры.
> - FN_ACC, ALT_ACC — специальные режимы акселератора (маскирование, наложение).
> - NMI — обычно отключён, вектор `0x0066`.
> - Страницы `#FE`, `#FC`, `#FD` — зарезервированы для системы.
> - Большинство этих регистров используется только BIOS/DSS; пользовательской программе они редко нужны напрямую.

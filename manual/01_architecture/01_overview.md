## Что такое ZX Sprinter

ZX Sprinter (торговое название — **Peters Plus Sp2000**) — советско-российский 8-битный компьютер,
разработанный компанией Peters Plus (Санкт-Петербург) в конце 1990-х — начале 2000-х годов.
Архитектурно это Z80-совместимый компьютер с аппаратной совместимостью с ZX Spectrum,
расширенный до полноценной мультимедийной платформы.

Основная идея: взять привычную Z80-экосистему и добавить всё, чего ей всегда не хватало —
большую память, быструю графику, жёсткий диск, стандартную DOS-подобную операционную систему.

### Altera ACEX 1K30 — главный чип

Выполняет **всю** нестандартную логику компьютера:
- Контроллер памяти с банкингом (DCP — Dynamic Configuration Port)
- Видеоконтроллер с тремя режимами
- Аппаратный блочный ускоритель (blitter)
- Синтезатор AY-3-8910 (эмуляция внутри FPGA)
- Цифровой аудио DAC (Covox, I²S)
- PS/2 клавиатура и мышь
- ISA-8 слот интерфейс
- Декодер портов I/O (256-входная LUT таблица)

Использует **91% логических ячеек** (1580 из 1728) и **57% RAM** (14080 бит).

### Altera MAX7000 — вспомогательный чип

- Контроллер FDD (WD1793 совместимый)
- VGA/SCART синхросигналы
- CMOS/RTC интерфейс
- Сброс и начальная инициализация

## Совместимость с ZX Spectrum

Sprinter поддерживает аппаратный режим ZX Spectrum 48K/128K:
- Стандартные порты `#FE`, `#7FFD`, `#BFFD`, `#FFFD`
- ZX-видеорежим (256×192, атрибуты)
- Pentagon и Scorpion-расширенное пространство страниц
- Большинство спектрумовского ПО работает без изменений

При переходе в нативный режим Sprinter получает:
- 4 МБ RAM вместо 128 КБ
- Линейный видеобуфер вместо ZX-экрана
- Нормальную файловую систему (FAT16) на HDD
- DSS — собственная DOS-подобная ОС

---

## Операционная система DSS

**DSS** (Disk SubSystem, версия 1.60) — резидентная ОС Sprinter:
- Загружается в защищённые страницы RAM при старте
- Предоставляет API через `RST #10` (50+ функций)
- FAT16 файловая система на IDE HDD
- Многозадачность не поддерживается, но процессы запускаются через `EXEC`
- Стандартный формат исполняемых файлов: `.EXE` с собственным заголовком

## BIOS

**BIOS** (версия 3.x) — встроен во Flash-память:
- Инициализирует FPGA (загружает битстрим из Flash)
- Предоставляет низкоуровневый API через `RST #08` и `RST #30`
- Включает SETUP-утилиту (конфигурация через CMOS)
- Поддерживает загрузку с HDD, FDD, ROM

## История версий прошивки FPGA

Прошивка FPGA (Altera ACEX) прошла несколько итераций:

| Версия | Дата | Особенности |
|--------|------|-------------|
| 2.13 | ~2001 | Встроенные ROM Spectrum, старая схема управления ROM |
| 2.17 | 2002 | Последняя ветки 2.xx. Исходники открыты в 2009 (Public Domain) |
| 3.00.253 | апр 2002 | **Официальный релиз SU1** — последний публичный от Peters Plus |
| 3.03.253 | фев 2003 | Не выпускался официально, опубликован в 2009 |
| 3.04.253 | июн 2003 | Скорректированы тайминги DRAM под Alliance VRAM |

Текущие открытые исходники собирают уровень **2.17/3.00** (источник: [`sprinter_bios/README.md`](#ссылки)).

## Ключевые моменты

> - Sprinter — это не просто «быстрый Spectrum»: это полноценный компьютер с HDD, DOS, 4+ МБ RAM и цветной графикой.
> - Вся нестандартная логика сосредоточена в двух Altera-чипах — понимание их работы = понимание Sprinter.
> - Программирование возможно на трёх уровнях: прямое железо (порты), BIOS API (RST #08), DSS API (RST #10).
> - Совместимость с ZX Spectrum сохранена — старый код работает, но нативные программы используют расширенные возможности.

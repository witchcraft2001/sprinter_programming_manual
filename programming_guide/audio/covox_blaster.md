# Covox-Blaster

Covox-Blaster (CBL) добавляет буфер на 256 байт и аппаратный счётчик (15/22 кГц). Процессор просто заполняет половины буфера, а ПЛИС самостоятельно выдаёт семплы на ЦАП. Статус текущей половины отображается в бите 7 порта `#FE`.

## Включение режима CBL

Порт `#4E` управляет режимами CBL. Для классической конфигурации достаточно записать `#80`.

```asm
        LD  BC,#4E
        LD  A,#80       ; включить буферизированный режим
        OUT (C),A
```

## Запись блоков

Счётчик автоматически переключает банки по 128 байт. Бит 7 порта `#FE` показывает активный банк (0 — нижняя половина, 1 — верхняя). Заполняем половину тогда, когда отображается другая.

```asm
; HL = адрес PCM, A = номер страницы (для PAGE3),
; каждая половина содержит 128 байт
FillCBLHalf:
        IN   A,(#FE)
        XOR  80h            ; выяснить, где проигрывается звук сейчас
        BIT  7,A
        JR   Z,FillCBLHalf  ; ждём переключение

        PUSH AF
        LD   A,(saved_page3)
        OUT  (#E2),A        ; восстановить, если нужно
        POP  AF

        LD   A,(pcm_page)
        OUT  (#E2),A        ; подключить страницу PCM в окно #C000

        LD   BC,#00FB
        LD   D,128
loop:
        OUTI
        DEC  D
        JR   NZ,loop

        LD   A,(saved_page3)
        OUT  (#E2),A
        RET
```

В ISR достаточно вызвать `FillCBLHalf` дважды за период (по переходам бита 7). При переполнении адреса страницы переключите `pcm_page` и `HL` на следующую страницу данных.

## Советы

- Если звук берётся из DSS страницы, заранее запросите её через `Dss.GetMem` и переключайте `PAGE3` только на время `OUTI`.
- Для стерео (когда появятся новые прошивки) используйте дополнительные биты порта `#4E`. В текущей версии всегда выводится моно.
- Чтобы «заглушить» CBL, заполните весь буфер значением `#80` (середина шкалы) и сбросьте страницу `SND_P` в ноль.

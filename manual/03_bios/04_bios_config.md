## CMOS RAM

CMOS — небольшой объём энергонезависимой памяти (~128 байт), питаемой от батарейки. Хранит настройки BIOS (SETUP), часы реального времени и параметры устройств.

В Sprinter CMOS доступен двумя способами:
1. **Через ISA-порты** (`#FFBD` чтение, `#BFBD` запись, `#DFBD` адрес).
2. **Напрямую через DCP** (порты `#1C`–`#1E` через MAX7000).

### Стандартный доступ к CMOS

```asm
CMOS_DRD equ    #FFBD       ; чтение данных
CMOS_DWR equ    #BFBD       ; запись данных
CMOS_AWR equ    #DFBD       ; запись адреса регистра

; Прочитать регистр CMOS #0E (флаги)
ReadCMOS:
        ld      bc, CMOS_AWR
        ld      a, #0E          ; номер регистра
        out     (c), a          ; выбрать регистр
        ld      bc, CMOS_DRD
        in      a, (c)          ; A = значение регистра
        ret

; Записать в регистр CMOS
WriteCMOS:
        push    af              ; A = значение
        ld      bc, CMOS_AWR
        ld      a, #0E          ; номер регистра
        out     (c), a
        pop     af
        ld      bc, CMOS_DWR
        out     (c), a
        ret
```

## Карта регистров CMOS

### RTC (часы реального времени)

| Регистр | Назначение |
|---------|-----------|
| `#00` | Секунды |
| `#02` | Минуты |
| `#04` | Часы |
| `#06` | День недели |
| `#07` | День месяца |
| `#08` | Месяц |
| `#09` | Год (младшие 2 цифры) |
| `#0A`–`#0D` | Статус RTC / Control |
| `#32` | Век (20/21) |

### Конфигурация BIOS (из DSETUP.ASM)

**Регистр `#0E` — флаги старта:**

| Бит | Маска | Имя | Описание |
|-----|-------|-----|----------|
| 7 | `#80` | MEMTEST | Тест памяти при старте |
| 6 | `#40` | SAFERAM | Safe RAM disks |
| 5 | `#20` | EXTBIOS | Обновление BIOS |
| 4–3 | `#18` | SDELAY | Задержка старта (0/250/500/750 мс) |
| 2 | `#04` | RUS | Язык (0=EN, 1=RU) |
| 0 | `#01` | QSTART | Быстрый старт |

**Регистр `#0F` — настройки клавиатуры:**

| Бит | Имя | Описание |
|-----|-----|----------|
| 7 | TMAT_ON | Автоповтор вкл/выкл |
| 6–5 | TDL | Задержка автоповтора (250/500/750/1000 мс) |
| 4–0 | TRATE | Скорость повтора (6/8/10/12/15/20/24/30 cps) |

**Регистр `#10` — выбор загрузочного диска:**

| Биты | Значение | Устройство |
|------|---------|-----------|
| [2:0] | SD | Системный диск: 0=FDD1, 1=FDD2, 2=IDE1, 3=IDE2, 4=RAM |
| [6:4] | ASD | Альтернативный: 0=FDD1, 1=FDD2, 2=IDE1, 3=IDE2, 4=RAM |

**Регистр `#11` — конфигурация приводов:**

| Биты | Значение |
|------|---------|
| [1:0] | FDD 1: 0=720KB, 1=1.44MB, 2=нет |
| [3:2] | FDD 2: те же значения |
| [5:4] | IDE Master: 0=AUTO, 1=SETUP, 2=DISABLED |
| [7:6] | IDE Slave: те же значения |

**Регистры `#12`–`#15` — параметры IDE Master (Setup-mode):**

| Регистр | Значение |
|---------|---------|
| `#12`–`#13` | Цилиндры (16 бит) |
| `#14` | Головки |
| `#15` | Секторы на трек |

**Регистры `#16`–`#19` — параметры IDE Slave** (аналогично).

**Регистр `#1A` — цветовая схема SETUP** (биты 3–0).

**Регистр `#1B` — конфигурация аппаратуры:**

| Бит | Имя | Описание |
|-----|-----|----------|
| 0 | TURBO | 1 = Turbo 21 МГц при старте |
| 1 | TB_EXCH | Turbo при обмене включён |
| 2 | PT_EXCH | Port exchange |
| 4–3 | COMP | Тип компьютера: 0=Sprinter, 1=Spectrum256, 2=Pentagon 128, 3=Other |
| 7–5 | — | Зарезервировано |

**Регистр `#1C` — режим сброса (Basic / TR-DOS):**

| Значение | Режим |
|---------|-------|
| 0 | Reset → Basic 128 |
| 1 | Reset → Basic 48 + 128 mem |
| 2 | Reset → TR-DOS + 128 mem |
| 3 | Reset → Expansion (не используется) |
| 4 | Reset → TR-DOS + 48 mem |
| 5 | Reset → Basic 48 |

**Регистр `#1D`:**

| Бит | Назначение |
|-----|-----------|
| 1 | Показать сообщение перезагрузки |
| 0 | Защита HDD от записи |

**Регистр `#1E` — маршрутизация TR-DOS дисков A–D:**

| Биты | Диск |
|------|------|
| [1:0] | TR-DOS A: 0=Default, 1=FDD, 2=HDD, 3=RAM-диск |
| [3:2] | TR-DOS B: — |
| [5:4] | TR-DOS C: — |
| [7:6] | TR-DOS D: — |

**Регистр `#3F` — контрольная сумма** настроек.

## Пример: чтение времени из RTC

```asm
CMOS_DRD    EQU #FFBD
CMOS_AWR    EQU #DFBD

; Прочитать 6 байт времени: сек, мин, час, день, месяц, год
ReadTime:
        ld      ix, time_buf
        ld      hl, time_regs
        ld      b, 6
.loop:
        ld      a, (hl)             ; номер регистра CMOS
        push    bc
        ld      bc, CMOS_AWR
        out     (c), a
        ld      bc, CMOS_DRD
        in      a, (c)
        pop     bc
        ld      (ix+0), a
        inc     ix
        inc     hl
        djnz    .loop
        ret

time_regs:  db  #00, #02, #04, #07, #08, #09
time_buf:   ds  6       ; сек, мин, час, день, месяц, год
```

## Пример: переключение turbo-режима через CMOS

> **Внимание:** Turbo переключается **аппаратно** (F12+Ctrl+Shift), а не через CMOS. CMOS регистр `#1B` бит 0 задаёт лишь состояние при включении.

```asm
; Установить Turbo по умолчанию при следующей загрузке
SetTurboDefault:
        ld      bc, CMOS_AWR
        ld      a, #1B
        out     (c), a
        ld      bc, CMOS_DRD
        in      a, (c)
        or      #01             ; бит 0 = Turbo ON
        push    af
        ld      bc, CMOS_AWR
        ld      a, #1B
        out     (c), a
        pop     af
        ld      bc, CMOS_DWR
        out     (c), a
        ret
```

## Контрольная сумма CMOS

BIOS при старте проверяет контрольную сумму настроек (`TCHEKSM` в DSETUP.ASM). Если сумма неверна — применяются значения по умолчанию (`SETDEFX`).

При любой записи в CMOS нужно пересчитать и обновить контрольную сумму. Иначе при следующей загрузке BIOS сбросит настройки.

```asm
; Пересчитать контрольную сумму (простая XOR или sum)
RecalcCMOSSum:
        ld      hl, #0E         ; первый настроечный регистр
        ld      b, #1E - #0E    ; диапазон #0E..#1D
        xor     a
.loop:
        push    af
        ld      bc, CMOS_AWR
        ld      a, l
        out     (c), a
        ld      bc, CMOS_DRD
        in      a, (c)
        pop     bc
        add     a, b            ; B содержит текущую сумму
        ld      b, a
        inc     hl
        djnz    .loop
        ; B = контрольная сумма
        ; Записать в #3F
        ; ...
        ret
```

## Ключевые моменты

> - CMOS хранит настройки BIOS + RTC.
> - Доступ через ISA-порты `#FFBD` (чтение), `#BFBD` (запись данных), `#DFBD` (запись адреса).
> - Регистр `#0E` — флаги старта, `#10` — загрузочный диск, `#1B` — turbo/тип ПК.
> - Регистры `#00`–`#09` — время (сек, мин, час, день, месяц, год).
> - Всегда обновляйте контрольную сумму `#3F` после записи.
> - Turbo-режим переключается клавиатурой, CMOS задаёт только состояние при старте.

# Covox (моно)

Обычный Covox подключён к портам `#FB` и `#4F`. Запись в любой из них выводит 8-битный уровень на оба канала TDA1543 (моно). Устройства, рассчитанные на классический ZX Covox, работают без изменений.

## Базовый вывод

```asm
; HL указывает на буфер PCM длиной в DE байт
; порт #FB выбран как выход
        LD  BC,#00FB
next:
        OUTI            ; OUT (C),(HL); HL++, B--
        JP  NZ,next
```

## Генерация сигнала без буфера

Можно синтезировать простейший тон напрямую:

```asm
        LD  BC,#00FB
        LD  E,0         ; уровень
loop:
        OUT (C),E
        LD  A,25        ; задержка
wait:   DEC A
        JR  NZ,wait
        XOR 20h
        LD  E,A
        JR  loop
```

## Рекомендации

- В режиме turbo пересылка `OTIR` работает быстрее, поэтому для стабильной частоты PCM добавляйте программные задержки или переключайтесь в нетурбо.
- Если приложение использует DSS/BIOS окна, сохраняйте страницу `PAGE3` перед тем, как временно переключить её для доступа к таблицам звука.
- Covox можно миксовать с AY: просто продолжайте писать в порт `#FB` параллельно проигрыванию AY-трека.

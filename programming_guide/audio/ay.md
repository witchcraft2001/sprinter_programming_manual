# AY-3-8910

В ПЛИС платы Sp2000 реализован полный аналог AY-3-8910/8912. Он программируется через стандартные ZX-порты: `#FFFD` (выбор регистра) и `#BFFD` (запись данных), поэтому все процедуры и трекеры для Spectrum работают без изменений.

## Регистры

- `R0/R1`, `R2/R3`, `R4/R5` — частота каналов A/B/C (12 бит на канал).
- `R6` — генератор шума.
- `R7` — маска управляющих битов (включение тонов/шума, I/O портов).
- `R8-R10` — громкость каналов.
- `R11/R12` — период огибающей, `R13` — форма огибающей.

## Минимальный пример

Ниже подаёт ноту «до» на канале A. Частота = 0x0AC8 (примерно 523 Гц при 1.7734 МГц).

```asm
; выбрать регистр 7 и разрешить тон канала A
        LD  BC,#FFFD
        LD  A,7
        OUT (C),A
        LD  BC,#BFFD
        LD  A,%00111110      ; тон A включён, все остальное выключено
        OUT (C),A
; установить период канала A (0x0AC8)
        LD  BC,#FFFD
        LD  A,0
        OUT (C),A
        LD  BC,#BFFD
        LD  A,#C8
        OUT (C),A
        LD  BC,#FFFD
        LD  A,1
        OUT (C),A
        LD  BC,#BFFD
        LD  A,#0A
        OUT (C),A
; громкость = 15
        LD  BC,#FFFD
        LD  A,8
        OUT (C),A
        LD  BC,#BFFD
        LD  A,15
        OUT (C),A
```

## Советы

- Перед использованием убедитесь, что конфигурация Sprinter содержит AY (во всех прошивках Sprinter-1/2 он доступен).
- В многозадачных приложениях (Estex/DSS) лучше зарезервировать канал через свою систему звука, чтобы не конфликтовать с ОС.
- Для фоновой музыки в C используйте готовые плееры (например, `wyz_play_music()` в SDK) — они управляют AY и переключают треки без вмешательства пользователя.

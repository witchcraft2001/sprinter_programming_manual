## ISA в Sprinter

Sprinter имеет **один ISA-8 слот** на плате, позволяющий подключать:
- COM/LPT адаптеры
- Сетевые карты (например NE2000)
- Звуковые карты SoundBlaster
- SCSI контроллеры
- Другие PC-периферийные устройства

ISA интерфейс реализован в FPGA ACEX 1K30.

---

## Адресация ISA

В Sprinter доступ к ISA-пространству осуществляется через:

| Метод | Диапазон | Назначение |
|-------|---------|-----------|
| Z80 I/O порты | `#C000`–`#FFFF` | ISA I/O пространство (алиасы) |
| Z80 memory | `#C000`–`#FFFF` (WIN3) | ISA memory пространство |
| Специальный порт | `#9FBD` (`Port.ISA`) | Установка 8-битной страницы ISA |

### Константы из SDK

```asm
Port.ISA        EQU #9FBD       ; базовый ISA порт
isa_adr_base    EQU #C000       ; базовый адрес ISA в памяти
```

---

## Модель доступа к ISA I/O

ISA-адрес формируется из:
1. Страницы (8 бит) — устанавливается через `Port.ISA`
2. Смещения (адрес в пространстве памяти Z80 в WIN3)

### Запись в ISA-порт

```asm
; Записать байт в ISA-порт #3F8 (COM1 data)
Port.ISA        EQU #9FBD

        ; Выбрать страницу ISA (старшие биты адреса)
        ld      a, #03              ; страница 3 (3F8/2 → offset 0xF8 на странице 3)
        out     (Port.ISA), a

        ; Записать в память WIN3 по смещению
        ld      a, 'A'
        ld      (#C0F8), a          ; эффективный ISA #03F8
```

> **Замечание:** точная схема маппинга зависит от прошивки FPGA и распределения страниц. Выше — упрощённое представление.

### Чтение

```asm
        ld      a, #03
        out     (Port.ISA), a
        ld      a, (#C0F8)          ; A = значение ISA порта
```

## CMOS как ISA-устройство

CMOS/RTC в Sprinter — ISA устройство:

```asm
CMOS_DRD        equ #FFBD           ; чтение данных
CMOS_DWR        equ #BFBD           ; запись данных
CMOS_AWR        equ #DFBD           ; запись адреса регистра
```

Эти константы — уже готовые ISA-адреса, доступные через `IN` / `OUT`:

```asm
; Прочитать регистр CMOS #0E
        ld      bc, CMOS_AWR
        ld      a, #0E
        out     (c), a
        ld      bc, CMOS_DRD
        in      a, (c)
```

## Типичное подключение устройства

Допустим, нужно работать с **сетевой картой NE2000** (ISA):

```asm
; Базовый адрес NE2000 обычно #300 или #280
; Регистры: #300 (Command), #301 (Page start), ...

NE2K_BASE       EQU #300

; Инициализация
NE2000_Init:
        ; Установить страницу ISA для #300-#3FF
        ld      a, (NE2K_BASE / 256)
        out     (Port.ISA), a

        ; Записать команду STOP + PAGE0
        ld      a, #21
        ld      (#C000 + (NE2K_BASE & #FF)), a

        ; ... далее настройка регистров ...
        ret
```

## DCP записи для ISA

Из `DCP.MIF` портов ISA:

```
30 : 7000 % slot 1 ports %
     7001 % slot 2 ports %
     7002 % slot 1 mem   %
     7003 % slot 2 mem   %
```

Эти значения означают: при обращении к портам `#30`-`#33` FPGA направляет данные на ISA-слот. TYPE=7 в DCP означает MAX7000-маршрутизация.

---

## Прерывания от ISA

ISA устройства могут генерировать прерывания (IRQ). Sprinter маршрутизирует их на INT линию Z80. Программа должна настроить IM2 или установить обработчик через BIOS.

---

## Ограничения ISA в Sprinter

1. **Только 8-битный ISA** — 16-битные карты не работают (или работают в 8-битном режиме).
2. **DMA** — не поддерживается стандартный PC DMA.
3. **Питание** — слот ограничен по потреблению.
4. **Тайминги** — ISA-циклы медленнее, чем у настоящего PC.
5. **Совместимость** — не все карты корректно работают с нестандартной шиной Sprinter.

---

## Проверенные карты

По сообществу Sprinter пользователей, работают:
- NE2000-совместимые сетевые карты (через адаптированные драйверы)
- Простые COM-порты (16550 UART)
- LPT-порты

Не работают или частично:
- ISA Sound Blaster (зависит от DMA)
- VGA-карты (конфликт с встроенным видео)

## SprinterNet+

Отдельный проект — сетевая карта **SprinterNet+**, использующая ISA-слот. Эмулятор `sprintem` содержит поддержку:

```asm
; DSS функция D0 — SprinterNet+
        ld      b, 0                ; подфункция
        ld      c, #D0
        rst     #10
```

## Пример: примитивный COM-порт

```asm
; Отправить байт через UART 16550 на ISA #3F8

UART_TX:
        ; Установить страницу ISA
        ld      a, 3                ; #300-#3FF
        out     (Port.ISA), a

        ; Ждать TX empty (LSR bit 5)
.wait:
        ld      a, (#C000 + #FD)    ; LSR (3F8 + 5)
        and     #20
        jr      z, .wait

        ; Записать байт
        ld      a, (byte_to_send)
        ld      (#C000 + #F8), a    ; THR
        ret

byte_to_send:   db 0
```

---

## Ключевые моменты

> - ISA-8 слот реализован в FPGA, доступен через Z80 I/O и память.
> - Страница ISA выбирается через `Port.ISA = #9FBD`.
> - `isa_adr_base = #C000` — базовый адрес в Z80 пространстве (обычно WIN3).
> - CMOS/RTC — частный случай ISA-устройства.
> - Только 8-битный ISA, без DMA.
> - Поддерживаются простые устройства: UART, сетевые карты.
